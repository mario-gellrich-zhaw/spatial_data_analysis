Required for the exercises:
- Token from https://openrouteservice.org

Why do we need a token?
This token is a key that allows to request the OpenRouteService server.

Steps:
1.) Create an account on https://openrouteservice.org
2.) Log in and go to Dashbord -> Dev Dashbord -> Tokens
3.) Create a token
4.) Copy and save the token to a .txt file in your working directory named 'ors_token.txt'
